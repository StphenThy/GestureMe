# Screenshots Folder

This folder contains screenshots of the GestureMe applications.

## How to Add Screenshots:

1. **Run the launcher:**
   ```
   python launcher.py
   ```

2. **Take screenshots:**
   - Press **Windows + Shift + S**
   - Select the window area
   - Save with these names:
     - `launcher.png` - Screenshot of the launcher UI
     - `air_canvas.png` - Screenshot of Air Canvas drawing
     - `music_controller.png` - Screenshot of Music Controller

3. **Add to GitHub:**
   ```
   git add screenshots/
   git commit -m "Add application screenshots"
   git push
   ```

The images will automatically appear in your README.md on GitHub!
